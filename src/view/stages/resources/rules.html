Régles du jeu

Cette section va présenter les règles du jeu Ataxx

Configuration de base du jeu

Le jeu a besoin d'au moins deux joueurs (humain et/ou ordinateur). Il se joue sur une grille carrée de dimension 7x7. Le but du jeu est d'avoir le maximum de pièces sur le plateau à la fin en convertissant le plus grand nombre de pièces adverses.

Chaque joueur commence avec deux jetons placés dans les coins du plateau. Le joueur bleu commence la partie

Règles

Comment jouer

Durant leur tour, les joueurs déplacent une de leur pièce d'un ou de deux espaces dans n'importe quelle condition (horizontal, vertical ou diagonale). Le déplacement ne peut cependant pas sauter un piè qu'elle lui appartienne ou qu'elle soit à l'adversaire. Si la destination est adjacente à la source, une nouvelle pièce est créée sur la case d'arrivée vide. Sinon, la pièce se déplace de la source vers la destination. Après le déplacement, toutes les piè de l'adversaire adjacentes sont converties en piè du joueur ayant fait le déplacement. Les joueurs alternent tant qu'un déplacement possible existe pour un joueur. Dans le cas contraire, il doit passer son tour.

Fin du jeu

Le jeu s'arrê quand toutes les cases ont été remplies ou un des joueurs n'a plus de pièces. Le joueur en ayant le plus a gagné. Il peut y avoir match nul quand le nombre de jetons conquit est égal.

Variations possibles

Nous avons décidé de mettre en place les variations possibles:

  • Commencer avec un ou deux jetons
  • Ajout de cases noires sur lesquelles on ne peut pas jouer
  • Taille variable de la grille entre 7x7 et 10x110
  • Plusieurs niveaux de difficulté possible
  • Possibilité de choisir l'algorithme de l'ordinateur
